# 《忆江南》AI互动微课项目

## 项目说明

本项目为吴老师参加“人工智能赋能教师成长培训”课程所完成的实践作业项目。

项目围绕小学语文古诗《忆江南》设计一节AI辅助互动微课，通过动画导入、古诗学习页面以及互动小游戏的形式，展示人工智能技术在语文课堂教学中的应用实践。

本项目教学设计、课程结构及教学内容均由吴老师独立完成。


---

## 微课学习流程

课程整体结构如下：

开场动画（8秒）  
↓  
古诗展示与朗读  
↓  
诗句理解  
↓  
互动小游戏  
↓  
完成学习

通过“情境导入—知识学习—互动练习”的方式，引导学生逐步理解古诗内容。

## 古诗原文

《忆江南》  
作者：白居易

江南好，  
风景旧曾谙。  

日出江花红胜火，  
春来江水绿如蓝。  

能不忆江南？
