《忆江南》AI互动微课项目

一、项目简介

本项目是一个基于AI工具辅助制作的小学语文互动微课，通过动画导入、古诗学习与互动小游戏相结合的方式，帮助学生理解唐代诗人白居易的经典诗词《忆江南》。

项目结合AI生成动画、网页技术与互动设计，让传统古诗词学习更加生动有趣，提升学生学习兴趣。

本项目主要用于展示AI在语文课堂教学中的辅助应用。


二、课程结构

整个微课流程如下：

开场动画（8秒）
↓
古诗展示与朗读
↓
诗句理解
↓
互动小游戏
↓
完成学习

通过“情境导入—知识学习—互动练习”的结构，让学生逐步理解古诗内容。


三、项目页面结构

index.html  
微课开场动画页面，用于展示江南水乡动画并引入课程。

poem.html  
古诗学习页面，展示《忆江南》原文并提供朗读功能。

game.html  
互动小游戏页面，通过小游戏形式巩固学生对诗句的理解。


四、古诗原文

忆江南  
作者：白居易

江南好，  
风景旧曾谙。  

日出江花红胜火，  
春来江水绿如蓝。  

能不忆江南？
